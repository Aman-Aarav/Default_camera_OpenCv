Webcam Stream using Qt and OpenCV

A simple real-time webcam streaming application built with C++, Qt, and OpenCV.
This project captures video frames from your systemâ€™s default webcam and displays them in a live window using OpenCV.

Features

    Opens and streams live video from the default webcam (VideoCapture(0)).
    Displays real-time frames in a window using cv::imshow.
    Exits gracefully when the ESC key is pressed.
    Demonstrates Qt + OpenCV integration inside a GUI project.

Requirements

     Qt 5 or Qt 6
     OpenCV 4.x or newer
     A working webcam
